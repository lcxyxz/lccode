# Plan: Replace Custom Input with ink-text-input

## Goal
Replace the custom `useIMEInput` + `InputLine` input system with `ink-text-input` package to get standard text editing behavior (cursor position tracking, left/right navigation, home/end support).

## Current Architecture
- `useIMEInput.ts` - Custom hook wrapping Ink's `useInput`, manages text buffer and key dispatch
- `InputLine.tsx` - Simple rendering component (prompt + text + blinking cursor)
- `useTerminal.ts` - Orchestrator connecting all hooks

## Changes Required

### 1. Install ink-text-input
```bash
npm install ink-text-input
```

### 2. Modify `src/frontend/hooks/useIMEInput.ts`
- Remove character input handling (lines 98-101: `setInput(prev => prev + char)`)
- Remove backspace handling (lines 91-95: `setInput(prev => prev.slice(0, -1))`)
- Remove space handling (lines 82-88: `setInput(prev => prev + ' ')`)
- Keep special key handling: Escape, Ctrl+C, Ctrl+L, arrows, Tab
- Accept `input` and `setInputValue` as parameters (synced from ink-text-input)
- Return `{ input, isComposing }` without internal text state management

### 3. Modify `src/frontend/components/InputLine.tsx`
- Import `TextInput` from `ink-text-input`
- Replace custom rendering with `<TextInput value={input} onChange={setInputValue} onSubmit={handleSubmit} />`
- Keep prompt icon and status display
- Handle overflow via container width

### 4. Update `src/frontend/useTerminal.ts`
- Pass `input` and `setInputValue` to `useIMEInput` (now from ink-text-input)
- Handle `onSubmit` callback for Enter key
- Sync state between ink-text-input and terminal state

## Files to Modify
1. `package.json` - Add ink-text-input dependency
2. `src/frontend/hooks/useIMEInput.ts` - Simplify to only handle special keys
3. `src/frontend/components/InputLine.tsx` - Use TextInput component
4. `src/frontend/useTerminal.ts` - Update input flow wiring

## Verification

2. Test special keys: Escape (mode toggle), Ctrl+C (exit), Ctrl+L (clear)
3. Test arrow keys: up/down for history, left/right for cursor movement
